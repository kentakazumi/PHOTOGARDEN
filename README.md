# POST Garden

## サイト概要
<ガーデニング画像投稿サイト>

### サイトテーマ
<植物、花好きな人が集まるコミュニティーサイト>

### テーマを選んだ理由
<自宅の庭の花の画像をいろんな人に見てもらったり、他の人がどのようなガーデニングをしているか見たかっため>

### ターゲットユーザ
<10代kから60代の植物、花が好きな人>

### 主な利用シーン
<自宅のガーデニングを撮影した時>
